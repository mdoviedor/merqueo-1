# Merqueo Test


## install
- Instalar python3
- Instalar pip3
- clonar el proyecto
  ``
    git clone ...
  ``
- Ubicarse en la carpeta del proyecto
``
  cd merqueo
``
- Install virtualenv
``
  pip3 install virtualenv
``
- Crear virtualenv
 ``
  virtualenv venv
 ``
 - Activar virtualenv
 ``
  source venv/bin/activate
 ``
 - Instalar requerimientos
 ``
  pip install -r requirements.txt
 ``
 - Iniciar el servicio
 ``
  python manage.py runserver 127.0.0.1:8000
 ``
